# Electronics Sales Dashboard (Excel)

## Overview
This project analyzes sales performance data for electronic gadgets using Microsoft Excel. The dashboard enables interactive exploration of sales trends, product performance, and regional distribution using pivot tables and slicers.

## Dataset
The dataset includes:
- Product category
- Product name
- Sales revenue
- Quantity sold
- Region
- Date

## Tools & Features Used
- Microsoft Excel
- Pivot tables
- Pivot charts
- Slicers
- KPI summaries
- Interactive dashboard design

## Analysis Performed
- Sales trend analysis over time
- Top-performing products and categories
- Regional sales comparison
- Quantity vs revenue insights

## Key Insights
- A small number of products contribute a large share of total revenue.
- Certain regions consistently outperform others.
- Seasonal patterns observed in gadget sales.
- Interactive slicers enable fast drill-down analysis.
